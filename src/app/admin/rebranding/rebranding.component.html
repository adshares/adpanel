Rebranding

This site presents all assets which are brand - dependent. They consist of icons and logos in different sizes for different screens.
To change images project needs to be rebuild. Collect all images which you want to change in one directory.
If you use interactive installer, you will be asked if you want to copy brand assets inside project directory. In case of automatic installer two environment variables should be set.
  • `COPY_BRAND_ASSETS` should be `Y`.
  • `BRAND_ASSETS_DIR` should be full path to assets directory.
Files from assets directory will overwrite the standard files.
Favicons
File favicon.ico should also be changed for legacy browsers.
